Version: 0.0.4 (Released on June 13, 2013)

Tested on OS X Mountain Lion, Ubuntu 13.04, Windows 8 and Windows 7.

Changelog

0.0.4 (June 13, 2013)

  • Windows support
  • Make texts, rectangles and ellipses
  • Workspace states (sidebar positions and colors) are saved on quitting
  • Some optimizations that improve brush response
  • New from clipboard image
  • More beautiful and faster redrawing of canvas when zoomed down
  • Several bug fixes and small improvements

0.0.3 (Feb 25, 2013)

  • Retina display support
  • Tablet pointer (Pen / Eraser) associated with brush preset
  • Brush preset associated with brush size

0.0.2 (Feb 23, 2013)

  • File format revised
  • Brush responce became far better - especially when you are using a Wacom tablet
  • The workspace feature added - you can open multiple main windows (workspaces)
  • Multiple views for one file
  • Several bug fixes

0.0.1 (Jan 18, 2013)

  • Initial release
