PaintField

An open-source digital painting / image editing app for Windows, Mac and Linux

Download »

PaintField is still a preview version!


PaintField is still very incomplete as a normal painting app.
Currently, the most suitable purpose of using PaintField is sketching.

PaintField aims to provide easy, pleasant and non-frustrating painting.
PaintField is designed to run fast and be as simple as possible.

Free and Free

PaintField is free of charge and open-source. The source code is released under MIT license.

Simple UI

The user interface of PaintField is very simplified and easy-to-use.

Painting with Wacom Tablets

PaintField is suitable for Wacom tablets.

Cross-platform

You can use PaintField on Mac, Linux and Windows.

Movable Tabs

You can move document or sidebar tabs around. Tabs can be either docked or floating.

Multiple Views

You can edit multiple documents or edit one document in multiple views at the same time.
