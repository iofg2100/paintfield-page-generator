Features

The current features of PaintField are:

  • Tabs
  • Split tab areas vertically / horizontally
  • Multiple views for one document
  • Navigator (Canvas rotation & scaling)
  • Layers with groups
  • Infinite undo / redo
  • Layer move
  • Simple brush, which can be used with Wacom tablets
  • Make texts, rectangles and ellipses
  • ...Too few features?
